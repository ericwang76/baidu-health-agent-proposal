模式本质：四方价值创造

对用户

  • 搜索即服务——不再需要自己判断"该不该看医生"
  • 零成本体验真人医生（机构补贴，用户永远免费）
  • AI预采集信息，医生接手即了解情况，体验远优于传统问诊
  • 从"信息焦虑"到"行动方案"一步到位

对医生

  • 只接高意向、已预筛的患者（不浪费时间在"随便问问"上）
  • 收到结构化病史摘要，30秒判断是否接诊
  • 确定性收入（机构保障诊费）+ 到店消费带来业绩
  • 获得百度搜索级别的精准患者流量

对医疗机构

  • 获客成本降低10倍+：¥50补贴 vs ¥500+广告点击，获得的是已建立信任的精准客户
  • 到店转化率3倍提升：30% vs 传统广告10%，因为用户已和机构医生建立关系
  • ROI可视化：平台提供补贴投入→到店→成交全链路数据看板
  • 从"买流量"到"买信任"：同样花钱，但用户从反感变为感激

对平台（百度）

  • 流量变现升级：从CPC广告（监管风险高、用户反感）升级为交易抽成+服务竞价（合规、可持续）
  • 零补贴成本运营：成熟期机构承担全部补贴，平台纯收手续费+竞价费
  • 数据飞轮：对话数据持续优化模型→转化率提升→机构ROI更高→愿意出更高价
  • 生态锁定：机构一旦习惯"按效果付费"获客，不会回到"按点击付费"老模式
  • 收入天花板打开：从医疗广告市场（有上限）进入医疗服务交易市场（无上限）

核心经济逻辑

Phase 1（平台补贴验证期）： 单次调度利润 = 交易抽成(GMV × 15-25%) - 医生补贴(¥30) - AI推理成本(≈¥0.1) Phase 3（机构补贴成熟期）： 单次调度利润 = 交易抽成(GMV × 15-25%) + 服务竞价费 - AI推理成本(≈¥0.1) 补贴成本 = ¥0（机构承担） 规模化利润 = Σ(单次利润) + 用户LTV增量（到店消费/复诊/慢病管理/药品复购）

关键洞察：

  • AI成本趋近于零：科普问答由AI承担，边际成本≈0，70%的流量免费服务完毕
  • 终态平台零补贴成本：机构将诊费补贴视为获客工具（等同广告费），平台只做匹配和抽成
  • 服务竞价是新收入引擎：多机构竞争同一用户时的出价费用，类似凤巢广告竞价但用户体验更好
  • 精准调度是利润核心：转化预测模型确保机构补贴花在"会到店"的用户身上，机构看到ROI才会持续投入
  • 到店消费拉高GMV：线上问诊客单¥150 vs 到店消费客单¥3000-30000，GMV量级跃升

🏢模式升级：机构补贴诊费 = 新型获客工具

核心洞察：把广告费变成用户服务

传统模式中，医疗健康机构在百度投放竞价广告获客。新模式下，机构将广告预算转化为旗下医生的诊费补贴——同样是花钱触达用户，但用户从"被打扰看广告"变为"获得免费医生服务"。

广告的本质是"花钱买注意力" → 新模式是"花钱买信任"
广告买到一次曝光机会 → 补贴买到一段医患关系

服务竞价机制（类广告竞价，但用户体验更好）

当多家机构的医生都符合调度条件时，引入服务竞价：

  • 机构可通过提高补贴金额获得优先调度权
  • 补贴越高 → 用户"免费"力度越大 → 接受率越高
  • 但平台同时考虑医生服务质量、匹配度，不是纯出价排序
  • 本质：把广告竞价商业模式用在服务竞价上，用户感知从"谁给钱多展示谁"变为"帮你匹配最合适的免费医生"
调度优先级 = α×机构出价 + β×医生服务质量 + γ×匹配度 + δ×用户接受概率

💊药企补贴：合规引入第三方费用

药企为什么有费用？

药企（尤其是处方药/OTC品牌药企）拥有庞大的学术推广预算和患者教育预算，这是行业公开的常规支出：

费用类型传统去向年行业规模痛点
学术推广费医药代表拜访、学术会议~3000亿/年合规压力大，效率低，无法量化ROI
患者教育费科普文章、患者手册、义诊活动~200亿/年触达有限，无法追踪效果
数字化营销费互联网医疗平台合作、医生在线教育~500亿/年缺少真实诊疗场景数据
DTP药房引流费线下推广、患者管理项目~100亿/年获取合规处方难、患者流失率高
核心认知：药企不缺钱，缺的是"合规+可量化+直达患者"的费用出口。 传统路径：药企 → 医药代表 → 医生 → 处方（合规灰色地带） 新路径：药企 → 平台学术服务费 → 补贴真实诊疗场景 → 患者受益（完全合规）

药企合规参与框架

关键原则：药企费用不直接与"开处方"挂钩，而是用于"患者教育"和"学术服务"——这是现行法规允许的合规路径。

《互联网诊疗监管细则》（2022）互联网诊疗中医师应独立诊疗医生自主诊疗，药企不得干预处方
国家医保局反商业贿赂禁止药企向医生直接利益输送本模式药企→平台→患者受益，医生无额外利益
《数据安全法》《个人信息保护法》个人数据处理需合法正当必要药企仅获脱敏统计，不触及个人信息

核心合规逻辑：药企买的是"服务"（患者教育触达、疾病数据统计），不是"处方"。平台是独立的学术服务商，不是药企的销售渠道。

排序因素

因素含义设计思路

探索加权新医生/新机构曝光Thompson Sampling

排序公式（融合服务竞价）

Score(doctor, user) = α×conversion_rate + β×speed_score + γ×price_fit + δ×quality_score + ε×bid_amount - λ×load_penalty 其中 bid_amount = 机构补贴出价（服务竞价维度） 最终排序 = 模型打分 × 质量门槛（质量不达标则不参与排序，防止纯出价驱动）

进阶方案：Contextual Bandit (LinUCB) 自动学习"哪类用户偏好哪类机构/医生"的非线性规律。

🏰核心竞争壁垒

流量垄断 + 意图纯度

百度搜索是中国最大的医疗信息入口，用户主动搜索意图纯度远高于信息流。竞对无法复制这个入口。

双边网络效应

更多机构入驻→更多科室覆盖→用户体验更好→更多用户使用→更多机构愿意投预算。先发者一旦启动飞轮，后来者追赶成本指数级增长。

数据飞轮

对话数据→优化转化模型→提升匹配精度→更高ROI→机构愿意出更高价→平台收入增长→投入更多研发。数据壁垒随时间累积。

凤巢基因复用

百度20年搜索广告经验（竞价排序、质量分、反作弊）可直接复用到"服务竞价"，这是其他互联网医疗平台不具备的能力。

合规护城河

互联网医院牌照+数据合规+医生备案体系一旦建立，后来者需要1-2年才能补齐。先行者占据监管信任优势。

从广告到交易的不可逆迁移

一旦机构习惯了"按效果付费"（补贴→到店成交），就不会再回到"按点击付费"的老模式。平台锁定效应极强。
